# FE-Switcheroo

A frontend switcher for Retropie, Pegasus and attract Mode on Pi4.

Version 1.00

# Install/Update 

open terminal type

curl -sSL https://bit.ly/3uHofsU | bash


# Attract Mode Usage
   
   -Exit attract mode
   
   -Type " switcheroo " , Press A

# Emulation Station Usage
  
   -Go to RetroPie/FE-Switcheroo, Press A
   
# Pegasus FE 

   -Go to RetroPie/FE-Switcheroo, Press A
